Step 1: Start a new HTML file

These are the HTML tags that all Web pages have.

They form the basic structure of the page.

  • The <!DOCTYPE html> tag tells the browser what kind of file this is.
  • The <head> tag is where you describe the page.

To begin:

  1. Open up your text editor (Text Wrangler or Notepad++) and start a new HTML file.
  2. Type the tags in the illustration above into your new file.
  3. Save the file as tags.html on your hard drive in the same place as your other INFO 101 files.

Step 2: Make a Head tag to describe your page

The head tag is where you setup the description of your web page such as the title of your page. You also link CSS stylesheets related to your web page in the head tags. We will talk about CSS more in a future activity.

  1. Inside the head tag, you give your web page a title by using the "title" tag. In the example above, < title > Taylor's Website < / title > names the page "Taylor's Website." Put at "title" tag inside your "head" tag and save your file.
  2. Navigate to the file you have created and open it locally in a browser.

The Body Tag Adds Content to Your Page

The < body > < / body > tag is where all the content that the user will see is placed. There are a lot of tags that can go inside the "body" tag. In this exercise we will study:

  • Heading (H) tags which add titles to the content.
  • Paragraph (P) tags which create paragraphs of text.
  • List (UL, OL, and LI) tags which add bullets and numbered lists to the content.
  • Anchor (A) tags which create links.
  • Division (DIV) tags that break the content up into blocks.
  • Image (IMG) tags that add pictures to the content.

Step 3: Add Heading (H) tags

  1. Type < h1 > [Your Name] < / h1 > in your html document, Save, and refresh your site in the browser. Notice the change.
  2. Under the h1 tag you created, create another header tag < h2 > About Me < / h2 > . Save the file and refresh your browser to notice how the h2 tag produced a smaller heading. You can use the heading tags from h1 (biggest) - h6 (smallest)
  3. Under the "About Me" heading, add at least 2 <p></p> tags. These create paragraphs. Inside each "p" tag, type a sentence or two to describe yourself.
  4. Add another h2 tag for "My Favorite Sites" and another for "My Favorite Pictures"
  5. Save and refresh your browser to see your headings.

Step 4: Create a List of Sites

Under the "My Favorite Websites" h2 tag, create a list of links to your favorite Websites.

  • You can create a numbered or "Ordered List" by using the < ol > < / ol > tag.
  • You can create a bullet or "Unordered List" by using < ul > < / ul >
  • In either case you can add lines to the list using < li > < / li > tags. "LI" is short for "list item."

To add a list to your page:

  1. Decide whether you want an ordered or unordered list. You can try one and then easily switch to the other kind by renaming the list tag.
  2. Create 5 list items with < li > < / li > tags.
  3. Within each li tag. Type the name a Website you like.
  4. Save your work, refresh the web page in your browser, and notice the changes on your website

Step 5: Make the site names into links to the sites

We're now going to allow our website users to click on one of your favorite websites and be navigated to that site using links.

  1. Inside each < li > < / li > tag type < a href="" > < / a >
  2. Inside the quotes of href="" type or paste the URL of your favorite site.
  3. Between the open and close "a" tag < a href="" > RIGHT HERE < / a > tag, type or paste the name of the Website.
  4. Save your file, refresh the web page in your browser, click all the links to make sure they work..

Step 6: Add a Set of Images

  1. Go online and find at least 2 images you like.
  2. Find and copy the URL of the image
    • In Internet Explorer, right-click on the image and choose "Copy Shortcut"
    • In Chrome, right-click on the image and choose "Copy Image URL"
    • In Firefox, right-click on the image and choose or "Copy Image Location"
  3. Add at least 2 images to your page under the "My Favorite Pictures" heading. To do so use the image tag : < img src="" title="" style="">
    • Inside the quotes in src="" paste the URL of the image.
    • Inside the quotes in title="" type a name for the image.
    • Inside the quotes in style="" type width:100px;height:100px;margin:5px. This will style the picture to be 100X100 pixels with a 5 pixel margin around it.
  4. Save your file, refresh your browser, and your pictures should now appear on the website.
  5. If one or more do not show up, make sure you go the URL correct.

Step 7: Make the images into links

Once you have at least 2 images on the page and at the right size, you can make them into links using an < a href="" > < / a > tag.

  1. Surround the "img" tag with an "a" tag like this < a href="" > <img src="" title="" style=""> < / a >
  2. Inside the quotes in href="" paste the URL of the page that the image was taken from.
  3. Save your file, refresh your browser, and your pictures are now links the Web sites where the pictures can be found.

What is CSS?

CSS stands for Cascading Style Sheets and it is used to create a look and feel for a web page.

What sorts of things are included in the term Look and Feel?

CSS consists of properties and values that are applied to an HTML tag:

  • The properties state what sort of look and feel you want to apply. For example you use the property "background-color" to apply a background color inside a tag
  • The values say what to do with the property. For example you can enter background-color:red to say that you want the background color to be red

Note that in the example above, a colon (:) separates the property from the value. That's the CSS syntax.

What does syntax mean? Look it up!

Many properties (like "background-color") are immediately understandable because they are in plain English. Many values (like 'red") are also easy to understand. Others are not.

What does the property "padding" do? Look it up!
